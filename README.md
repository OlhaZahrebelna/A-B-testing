# A/B Testing Analysis

## Project Overview

This project presents an A/B testing analysis conducted on a dataset related to an online marketing campaign. The primary goal is to evaluate the effectiveness of a new campaign (Test Campaign) compared to an existing one (Control Campaign). The analysis includes statistical tests and visualizations to assess key performance metrics.

## Dataset Description

The dataset consists of two groups:

  * **Control Group**: Represents users exposed to the existing campaign.

  * **Test Group**: Represents users exposed to the new campaign.

Key metrics analyzed:

  * Click-through Rate (CTR)

  * Content View Rate

  * Conversion Rate

  * Impression Cost

  * Purchase Cost

## Analysis Approach

1. **Data Cleaning & Preprocessing**: Ensuring data consistency and handling missing values.

2. **Exploratory Data Analysis (EDA)**: Visualizing distributions and relationships between key metrics.

3. **Normality Testing**: Using histograms, Q-Q plots, and the Shapiro-Wilk test to check normality assumptions.

4. **Statistical Testing**: Performing a t-test to compare means between the control and test groups.

5. **Result Interpretation**: Identifying statistically significant differences and drawing conclusions.

## Key Findings

  * The test group showed statistically significant improvements in CTR, content views, conversions, and impression costs.

  * No significant difference was found in purchase costs between the two groups.

  * The test campaign positively impacted most performance metrics but requires cost optimization.

## Recommendations

Consider implementing the test campaign for better engagement and conversion rates.

Further optimize ad spending to reduce impression and purchase costs.

Conduct additional experiments to refine targeting and maximize ROI.

## Conclusion

This analysis demonstrates the power of A/B testing in optimizing marketing campaigns. The findings highlight areas of improvement and provide data-driven recommendations for campaign adjustments.
